# Machine Monitoring Project

Real-time monitoring system for factory machines across multiple floors using Cloudflare Tunnel and Google Apps Script.

## Features
- Monitor 40+ machines across 3 floors
- Cloudflare Tunnel for secure public access
- Google Sheets integration for data storage
- Auto-reconnect and health monitoring
- Optimized for bandwidth and speed

## Tech Stack
- **Backend:** Node.js + Express
- **Proxy:** http-proxy-middleware
- **Tunnel:** Cloudflare Tunnel (Free)
- **Database:** Google Sheets via Apps Script
- **Monitoring:** Floor 1, Floor 2, Floor 3
